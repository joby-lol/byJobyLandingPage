I make web tools

I like my front ends lightweight, high performance, and easy to integrate. Also open source.

WebPasswordStrength

Suite of javascript tools for analyzing passwords and displaying widgets in forms.

onlySlider

An extremely lightweight carousel Javascript that supports any content HTML does, degrades gracefully, and actually supports bandwidth-saving through media loading control.

onlyVideo

A javascript tool for progressively enhancing hosted video links and embeds. Useful for making embedded videos visually consistent, and also greatly reduces browser draw times.

This is just the new stuff I'm excited about right now. Check out my GitHub account for even more stuff.
